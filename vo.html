Visual Odometry
This project was a part of AUAV[Autonomous Unmanned Aerial Vehicle project]. Visual Odometry is a problem of pose estimation of the vehicle from its video-feed. This project was an application of "Andrew Howard, Real time visual odometry for Autonomous Ground vehicles."
Pose estimation was done with two different sensors namely Microsoft Kinect and Stereo camera rig.
Microsoft Kinect's algorithm is slightly different from Stereo camera counter-part.
This paper briefs further the algorithm and results.
