Guide: Dr. Bharat Singh Rajpurohit, Dr. Suman Kalyan Pal.
What if we can transform the projector screen into a touch screen? A technology which goes by the name "Smartboards" enables the presenter to interact with the board but is extremely costly. This project is an attempt to enable the same but at a much lesser cost.
The system only required a computing device, webcam, a small laser light, projector and a flat surface for projection. The system enables click, double-click, scroll and drawing.
Below is the link to a short video that demonstrates the basic features of the system. Observe that the person in the video was interacting with the system from the projection surface.

The project involved
  • Calibration for the transformation matrix between the projector coordinates and the camera coordinates.
  • Laser-tip/red light detection and thresholding.
  • Event (click, double-click e.t.c.) detection and triggering of the appropriate event.
